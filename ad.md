# 카카오 애드핏(AdFit) 연동 가이드

## 1. 광고 추천 영역

Loglingo 앱 구조와 애드핏 지원 단위를 고려하여 다음 3곳을 추천합니다.

### A. 사이드바 광고 (PC/모바일 메뉴)
-   **위치**: 왼쪽 사이드바 네비게이션 최하단
-   **유형**: `250x250` (정사각형)
-   **선정 이유**: PC에서는 디자인을 해치지 않으면서 빈 공간을 활용하고, 모바일에서는 메뉴 하단에 자연스럽게 노출됩니다.
-   **추천 규격**: **250x250** (반응형 대응이 가장 용이함)

### B. 일기 목록 인피드 광고 (중간 삽입)
-   **위치**: 일기 목록 사이사이 (예: 5번째 항목 마다)
-   **유형**: `728x90` (PC) / `320x100` (모바일)
-   **선정 이유**: 사용자가 과거 일기를 보러 스크롤할 때 자연스럽게 시선이 머무는 곳입니다.
-   **추천 규격**: 반응형 분기 처리 (모바일 **320x100**, PC **728x90**)

### C. 일기 상세 하단 광고
-   **위치**: 일기 본문 카드 끝난 후, 학습 아카이브 섹션 전
-   **유형**: `320x100` (모바일) / `728x90` (PC)
-   **선정 이유**: 글을 다 읽은 시점에 노출되어 클릭률이 높습니다.
-   **추천 규격**: **320x100** (모바일 최적화 우선)

---

## 2. 구현 요청 프롬프트 (그대로 복사해서 사용하세요)

*아래 내용을 복사해서 AI에게 구현을 요청하세요.*

```markdown
# 카카오 애드핏(AdFit) 연동 요청

Loglingo 애플리케이션에 카카오 애드핏 배너를 구현해주세요. 아래 명세를 따라주세요.

## 1. `AdFitBanner` 컴포넌트 생성
`src/components/ads/AdFitBanner.tsx` 파일을 생성합니다.
- **Props**: `unitId` (string), `width` (number), `height` (number), `title` (접근성용, 선택사항).
- **구현 내용**:
  - `<ins class="kakao_ad_area" ...>` 요소를 렌더링합니다.
  - `useEffect` 내에서 스크립트 `//t1.daumcdn.net/kas/static/ba.min.js`를 동적으로 로드합니다.
  - 스크립트가 배너당 한 번만 실행되도록 처리합니다.
  - 페이지 이동 시 `window.kakao_ad_context` 등을 초기화하거나 재로드하는 로직을 고려하여 SPA 환경에서 광고가 갱신되도록 합니다.
  - 초기 로드 시 레이아웃 시프트를 방지하기 위한 스타일링을 적용합니다.

## 2. 광고 삽입 위치

### 위치 A: 사이드바 (네비게이션)
- **파일**: `src/components/layout/Navigation.tsx` (또는 해당 사이드바 컴포넌트).
- **작업**: 사이드바 메뉴 리스트 최하단에 `AdFitBanner`를 추가합니다.
- **Unit ID**: 플레이스홀더 `SIDEBAR_UNIT_ID` 사용.
- **크기**: 250x250.

### 위치 B: 일기 목록 (인피드)
- **파일**: `src/app/[locale]/diary/page.tsx`
- **작업**: 일기 목록 렌더링 시, 5번째 항목마다 광고 컴포넌트를 삽입합니다.
- **Unit ID**: 플레이스홀더 `LIST_UNIT_ID` 사용.
- **크기**: 모바일 친화적인 **320x100**을 기본으로 사용합니다.

### 위치 C: 일기 상세 (본문 하단)
- **파일**: `src/app/[locale]/diary/[id]/page.tsx`
- **작업**: 일기 내용 카드와 학습 아카이브 카드 사이에 광고를 삽입합니다.
- **Unit ID**: 플레이스홀더 `DETAIL_UNIT_ID` 사용.
- **크기**: 320x100.

## 3. 기타 요구사항
- Content Security Policy (CSP) 설정이 있다면 애드핏 스크립트를 허용하도록 수정해주세요.
- `src/config/ads.ts` 파일을 만들어 Unit ID 상수들을 관리해주세요.
```
